# LGU Reservation Dashboard

Simple Bootstrap + Chart.js dashboard with an Express API serving reservation data.

## Requirements

- Node.js 16+

## Install

```bash
npm install
```

## Run Locally

```bash
npm start
```

Then open:

http://localhost:3000
